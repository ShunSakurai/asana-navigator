Asana Navigator Updates

Thank you for using Asana Navigator!

Updates

I made some important updates:

  • Asana Navigator now requires an extra permission ("tabs" in manifest.json) to be compatible with Firefox. The behavior itself doesn't change. Please accept the elevated permission to continue using it. Thank you!
  • I started this Asana Navigator project because I didn't really like the change to push down the project list. I'm happily dropping the feature to display the project names on top of the task pane, because of the improved task pane layout. I'm grateful to the Asana team for listening to and valuing our voices :)
  • While I love Asana's new task layout on the right pane, some features didn't work with the new layout. I fixed the issues and all features should be fully functional now.

Reminder

  • Each feature can be switched on/off in the extension options.
  • Please try reloading the Asana task page if this extension doesn't work well.
