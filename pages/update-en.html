Asana Navigator Updates

Thank you for using Asana Navigator!

Updates

I made some important updates:

  • The number of text replacement is displayed on the toast notification (for user-defined strings). I also fixed a bug with "Use Preset Cleanup" feature.
  • I added keyboard shortcuts to attach files (Tab+V and Tab+1/2/3/4/5).
  • I also added a keyboard shortcut to open more actions (Tab+.).

Reminder

  • Each feature can be switched on/off in the extension options.
  • Please try reloading the Asana task page if this extension doesn't work well.
