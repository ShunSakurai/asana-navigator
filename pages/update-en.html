Asana Navigator Updates

Thank you for using Asana Navigator!

Updates

I made some important updates:

  • When you open the search bar, you now see the shortcut to search in the currently open project, tag, my tasks, or other users' tasks. You can benefit from this feature if "Advanced Search" is available in your Organization/Workspace. If not, it is recommended to turn off this feature from the extension options.

Reminder

  • Each feature can be switched on/off in the extension options.
  • Please try reloading the Asana task page if this extension doesn't work well.
