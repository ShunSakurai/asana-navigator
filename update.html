Asana Navigator Update

Thank you for using Asana Navigator!

Updates

I made some important updates:

  • The keyboard shortcut Tab+O (convert to subtask) has been changed to Tab+G. It's because Tab+O is now used to show/hide sidebar.
  • The extension now works more stable than ever, especially in displaying arrows to go to sibling subtasks.

Reminder

  • Each feature can be switched on/off in the extension options.
  • Please try reloading the Asana task page if this extension doesn't work well.
