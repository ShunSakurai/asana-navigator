Updates

Thank you for using Asana Navigator!
I made an important update. Please reload the Asana task page if this extension doesn't work well just after the update.

I made the following two changes:

  • A new feature is added to jump to any other sibling subtasks. You can now display a drop-down list containing all subtasks of the parent task.
  • The keyboard shortcut Tab+K (go to previous subtask) is changed to Tab+G. It's because Tab+K is now used to like/heart the task.
