Asana Navigator Updates

Thank you for using Asana Navigator!
I made some important updates. Please reload the Asana task page if this extension doesn't work well just after the update.

I made the following changes:

  • Pressing the keyboard shortcut Tab+E or clicking [More actions (...)] > [Replace Text in Description...] now opens a dialog view which allows you to see what you are replacing.
  • Added the feature to replace cutom text in the task description. It's also accessible through the keyboard shortcut Tab+E or by clicking [More actions (...)] > [Replace Text in Description...].
