Asana Navigator Update

Thank you for using Asana Navigator!

Updates

I made some important updates:

  • The keyboard shortcut Tab+N (show drop-down list of sibling subtasks) is changed to Shift+Tab+→. It's because Tab+N is now used to add subtasks.
  • The keyboard shortcut Tab+J/K (jump to sibling subtasks) is changed to Shift+Tab+↑/↓ accordingly.
  • The keyboard shortcut Tab+R (convert to subtask) is changed to Tab+O. It's because Tab+R is now used to jump to task description.

Reminder

  • Each feature can be switched on/off in the extension options.
  • Please try reloading the Asana task page if this extension doesn't work well.
