Asana Navigator Updates

Thank you for using Asana Navigator!

I made some important updates:

  • The keyboard shortcut Tab+N (open dropdown list of sibling subtasks) is changed to Shift+Tab+→. It's because Tab+N is now used to add subtasks.
  • The keyboard shortcut Tab+J/K (go to sibling subtasks) is changed to Tab+↑/↓ accordingly.
  • The keyboard shortcut Tab+R (convert to subtask) is changed to Shift+Tab+→. It's because Tab+R is now used to jump to task description.
